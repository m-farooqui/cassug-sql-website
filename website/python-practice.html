Practice Ideas

  1. Use df.head() to inspect records.
  2. Use df.shape to count rows and columns.
  3. Use df.info() to review data types.
  4. Use df.isna().sum() to check missing values.
  5. Use value_counts() for category counts.
  6. Use groupby() for analyst summaries.
  7. Create calculated columns.
  8. Build charts with Matplotlib, Seaborn, or Plotly.
  9. Parse simple HTML with BeautifulSoup.
  10. Train starter models with scikit-learn.
