Hands-On Practice Area

This section helps members practice real-world data skills directly from the website. You can practice SQL, analyze datasets with Python, build interactive visuals with Plotly, train beginner machine learning models, and later use Metabase for a BI-style dashboarding experience.

SQL Practice, Python Practice, Visualization Practice, Machine Learning Practice, and Metabase open in separate browser tabs so users can keep the main CASSUG website open.

SQL Practice Lab

Practice SQL in a browser-based SQL interface. The SQL lab uses temporary database copies, so users can run queries and practice safely without changing the original master database files.

  • Choose datasets by domain
  • Run SELECT, GROUP BY, JOIN, and filtering queries
  • Explore schema and table columns
  • Reset your temporary database copy when needed

Suggested Practice by Tool

Suggested SQL Practice

Try questions like:

  • Count records by category or domain-specific group
  • Find top 10 products, companies, precincts, programs, or regions
  • Use WHERE to filter records
  • Use GROUP BY to summarize data
  • Use ORDER BY to rank results
  • Create temporary analysis tables inside your practice copy

Suggested Python Practice

Try analysis like:

  • Use df.head() to inspect rows
  • Use df.shape to count rows and columns
  • Use df.isna().sum() to check missing values
  • Use value_counts() for category counts
  • Use groupby() to create summaries
  • Create calculated columns for analysis

Suggested Visualization Practice

Try building visuals with:

  • KPI cards for total counts, revenue, averages, or rates
  • Bar charts by category
  • Line charts by date or year
  • Pie or donut charts for share of total
  • Scatter plots for numeric relationships
  • Heatmaps for two-category comparisons

Suggested Machine Learning Practice

Try modeling tasks like:

  • Predict a category using classification
  • Predict a numeric value using regression
  • Choose a target column and feature columns
  • Compare model performance using common metrics
  • Review sample predictions against actual values
  • Discuss whether the dataset is appropriate for machine learning

Suggested Dashboard Practice

Try building dashboards with:

  • Total count card
  • Bar chart by category
  • Line chart by time period
  • Top 10 ranking table
  • Filters by year, state, borough, product, retailer, or class type
  • Map visuals where latitude and longitude are available

Learning Path

1. Start with SQL

Use SQL Practice to understand table structure, explore rows, filter records, group data, and answer business-style questions.

2. Analyze with Python

Use Python Practice to create summaries, clean columns, calculate trends, check missing data, and prepare deeper analysis.

3. Visualize with Plotly

Use Visualization Practice to turn your SQL and Python-style summaries into interactive charts like bar charts, line charts, scatter plots, heatmaps, and KPI cards.

4. Practice Machine Learning

Use Machine Learning Practice to choose targets, select features, train beginner models, and review classification or regression metrics.

5. Build BI Dashboards Later

Use Metabase later as a separate dashboarding practice tool when you are ready to bring back the full BI-style visual analytics environment.
