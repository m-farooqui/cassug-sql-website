Suggested Practice by Tool

Suggested SQL Practice

Try questions like:

  • Count records by category or domain-specific group
  • Find top 10 products, companies, precincts, or programs
  • Use WHERE to filter records
  • Use GROUP BY to summarize data
  • Use ORDER BY to rank results
  • Create temporary analysis tables

Suggested Python Practice

Try analysis like:

  • Use df.head() to inspect rows
  • Use df.shape to count rows and columns
  • Use df.isna().sum() to check missing values
  • Use value_counts() for category counts
  • Use groupby() to create summaries
  • Create calculated columns for analysis

Suggested Dashboard Practice

Try building dashboards with:

  • Total count card
  • Bar chart by category
  • Line chart by time period
  • Top 10 ranking table
  • Filters by year, state, borough, product, or class type
  • Map visuals where latitude and longitude are available

Learning Path

1. Start with SQL

Use SQL Practice to understand table structure, explore rows, filter records, group data, and answer business-style questions.

2. Analyze with Python

Use Python Practice to create summaries, clean columns, calculate trends, check missing data, and prepare deeper analysis.

3. Visualize with Metabase

Use Metabase to turn SQL and Python-style analysis into charts and dashboards that are easier for non-technical users to understand.
